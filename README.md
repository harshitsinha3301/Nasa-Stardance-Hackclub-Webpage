# 🚀 Cosmic Intelligence Portal

A NASA-inspired futuristic space exploration dashboard built for the **NASA Stardance Challenge** using pure HTML, CSS, and JavaScript.

## 🌌 Overview

Cosmic Intelligence Portal is an interactive mission-control style web application designed to provide an immersive space exploration experience. The project combines scientific information, modern UI design, and dynamic animations into a single responsive platform.

Built entirely without external libraries, APIs, frameworks, or image assets.

---

## ✨ Features

* Animated deep-space starfield
* Shooting meteor effects
* Real-time UTC clock
* Dynamic space statistics counters
* Solar System analytics dashboard
* Interactive planet information cards
* NASA mission archive section
* Space research center
* Glassmorphism user interface
* Scroll reveal animations
* Responsive design
* Modern aerospace-inspired visuals

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Canvas API
